# ?? Chandan's Interactive 3D Portfolio

Hey there! ?? Welcome to the repository for my personal portfolio. This is a highly interactive, 3D web experience designed to showcase my journey, projects, and skills as a Software Engineer. 

Feel free to explore the codebase, discover how the 3D animations work, and get inspired for your own creations! ?

## ??? Tech Stack
- **Core:** React, TypeScript, Vite
- **3D & Magic:** Three.js (React Three Fiber), GSAP, WebGL
- **Design:** Aesthetic Glassmorphism UI & Custom CSS
- **Deployment:** Vercel

## ?? Running Locally
Want to spin this up on your machine? It's super simple:

```bash
npm install --legacy-peer-deps
npm run dev
```
